Descripcion

desbloqueame es un híbrido de Mastermind y el patrón de bloqueo de Android. Un juego que te encantará odiar.
Esperemos que este juego (base de código incluida) te vuelva loco.
El objetivo de este proyecto era descubrir aplicaciones web progresivas con trabajadores de servicios y jugar con Webpack. También para entretener porque el tubo es bastante aburrido, especialmente la línea central en la hora pico.
Si te gusta este juego, debes convencer a 3 personas de que les guste, quienes tendrán que convencer a otras 3 personas ... y este proyecto se convertirá en la primera secta OSS. Además, si lo odias, debes convencer a 3 personas para que lo odien, quienes tendrán que convencer a otras 3 personas ...
